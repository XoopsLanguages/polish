Pomoc: Prywatne Wiadomości

OPIS

Moduł PW służy do prywatnego przesyłania wiadomości między użytkownikami

INSTALACJA/ODINSTALOWANIE

Nie są wymagane żadne specjalne środki, postępuj zgodnie ze standardowym procesem instalacji – rozpakuj folder /pm do katalogu ../modules. Zainstaluj moduł przez Admin -> Moduł Systemowy -> Moduły.

Szczegółowe instrukcje dotyczące instalacji modułów są dostępne w Podręczniku Operacyjnym XOOPS

INSTRUKCJA OBSŁUGI

Aby skonfigurować ten moduł, musisz:

  • Wejść do swojej Skrzynki Odbiorczej bezpośrednio ze strony głównej lub klikając górny link „Idź do modułu” w obszarze administracyjnym PW.
  • Skonfigurować swoje preferencje dla modułu (patrz „Preferencje”) i opcjonalnie blok PW, jeśli zamierzasz go używać (patrz „Bloki”).
  • Możesz masowo usuwać swoje wiadomości, korzystając z zakładki „Oczyszczanie” Wiadomości.
