Pomoc: Profil

OPIS

Moduł Profil służy do zarządzania niestandardowymi polami profilu użytkownika.

INSTALACJA/DEINSTALACJA

Nie są wymagane żadne specjalne środki, postępuj zgodnie ze standardowym procesem instalacji – rozpakuj folder /profile do katalogu ../modules. Zainstaluj moduł przez Admin -> Moduł systemowy -> Moduły.

Szczegółowe instrukcje dotyczące instalacji modułów są dostępne w Podręczniku operacyjnym XOOPS

INSTRUKCJE OBSŁUGI

Aby skonfigurować ten moduł, musisz:

  • Skonfigurować preferencje modułu (patrz „Preferencje”) oraz opcjonalnie blok Profil, jeśli zamierzasz go używać (patrz „Bloki”).
  • Edytować istniejące kategorie lub dodać nowe.
  • Edytować istniejące pola lub dodać nowe. Tutaj możesz określić, które pola będą widoczne w której kategorii oraz czy będą widoczne podczas rejestracji użytkownika.
  • Zdefiniować kolejność kroków rejestracji.
  • Na koniec możesz ustawić uprawnienia dla poszczególnych pól – które są edytowalne, które można przeszukiwać.

Środki antyspamowe

Aby zminimalizować rejestracje spamowe, wykonaj następujące kroki:

  • Przejdź do modułu Protector w panelu administracyjnym, przejdź do Preferencji, a następnie na dole, przy tej opcji: „Stop Forum Spam” Sprawdza dane POST w odniesieniu do spamerów zarejestrowanych w bazie danych www.stopforumspam.com.
    Ustaw na „Zablokuj IP (bez ograniczeń)”

  • W pliku /class/captcha/config.php upewnij się, że tryb jest ustawiony na „text”:

    return $config = array(
    'disabled' => false, // Wyłącz CAPTCHA
    'mode' => 'text', // domyślny tryb, możesz wybrać 'text', 'image', 'recaptcha' (wymaga klucza API)
    'name' => 'xoopscaptcha', // nazwa CAPTCHA
    'skipmember' => true, // Pomiń sprawdzanie CAPTCHA dla członków
    'maxattempts' => 10, // Maksymalna liczba prób na sesję
    );

  • W module Profil przejdź do panelu administracyjnego i w kroku podstawowym znajdującym się pod adresem:
    /modules/profile/admin/step.php?id=1
    ustaw opcję „Zapisz po zatrzymaniu” na „Nie”


  • W Preferencjach modułu Profil ustaw opcję „Użyj CAPTCHA po drugim kroku rejestracji” na „Tak” (jest to ustawienie domyślne)
